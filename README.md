# Tropical Cyclone Tangential Wind Decomposition (TC_WaveDecomp)


## Integrared with CyTrack Vortex Outputs

This repository provdes a high-performance Python framework to decompose Tropical Cyclone (TC) tangential wind fields into azimuthal wavenumbers (WN0, WN1, and WN2). The tool is specifically designed to ingest vortex center and structural data generated by CyTrack.

By transform WRF model output into a storm-centered cylindrical coordinate system, this utility allows researchers to isolate the symmetric vortex from asymmetric features.

### Key features
- CyTrack integration: Native support for CyTrack-formatted CSV/txt files (utilizing latc, lonc, Pc, Vmax, and ROCI).
- Vertical Profiling: Automated interpolation of WRF staggered C-grids to user-defined height levels (AGL).
- Fourier Reconstruction: WN0 - Axisymmetric mean (vortex intensity); WN1 - Dipole asymmetry (vortex tilt/motion); WN2 - Quadrupole asymmetry (elliptical structures/inner-core dynamics)
- Scalable Radius: Analysis domain is dynamically scaled to the Radius of Outer Closed Isobar (ROCI) provided by CyTrack.
- Multiprocessing: Built with ProcessPoolExecutor to handle large climate datasets across multiple CPU/GPU cores efficiently.
